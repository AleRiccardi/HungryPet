# Raspberry 
This is the project section for the Rasperry hungry pet.

## Type of messages
List of all possible type of message transferred.

#### Wifi connection
* `en : entity`
    * `bt : bluetooth`
    * `eng : engine`
* `ac : action`
    * `wg : wifi-get`
    * `ws : wifi-set`
* `cn : content`
    * `ssid : ssid`
    
#### Food Level
* `ac : action`
    * `bwl: bowl_level`
    * `cnl: container_level`
* `cn : content`
